# MCSaveHelper

一个管理 Minecraft 存档的工具。

## 这个工具能做什么

- **存档浏览器**：浏览和分析 Minecraft 存档，快速查看存档信息、玩家数据、区域文件和 NBT 数据
- 帮你转换存档的 UUID（比如换号、离线改名的时候用）
- 可以一个一个处理存档，也可以批量处理
- 支持自定义玩家和 UUID 的对应关系
- 能检测存档版本，也能清理存档里的垃圾文件（缓存、日志这些）

## 怎么用

### 你需要先有这些

- Python 3.9 或更新的版本
- Windows、Mac 或 Linux 都能用

### 安装步骤

1. 下载代码到本地
2. 进入项目目录
3. 安装需要的库：
   ```bash
   pip install -r requirements.txt
   ```
4. 运行程序：
   ```bash
   python main.py
   ```

## 主要功能

### 存档浏览器（新增）

探索和查看 Minecraft 存档的详细信息：

- **存档信息**：查看存档名称、游戏版本、游戏模式、出生点坐标
- **存档统计**：快速了解玩家数量和区域文件数量
- **玩家数据**：
  - 查看玩家状态（生命值、饥饿值、经验等级、氧气、坐标）
  - 浏览玩家物品栏
  - 查看玩家的完整 NBT 数据
- **区域热力图**：可视化存档的区域文件，根据文件大小着色
- **NBT 查看器**：
  - 树状浏览 NBT 数据结构
  - 搜索功能：快速定位关键字
  - 导出功能：将 NBT 数据导出为 JSON 格式

### 批量处理

- 选一个文件夹，程序会自动找到里面所有的存档
- 可以一次处理多个，不用一个个点
- 能实时看到处理进度

### UUID 映射管理

- 可以自己设置玩家名对应的 UUID
- 支持导入和导出
- 会帮你检查 UUID 格式对不对

### 其他功能

- 自动检测存档是哪个版本的
- 可以不联网处理（离线模式）
- 清理存档里没用的文件

## 项目结构

简单说一下：
- `app/` 是应用层，管界面和服务
  - `ui/views/` 包含所有视图组件，包括新的存档浏览器
- `core/` 是核心，处理存档的实际逻辑
  - `omni/` 包含存档会话管理
- `translations/` 是多语言文件
